## Cleaned Data

This folder contains the cleaned and prepared datasets used in this report.

- `atp_matches.rds`: contains information regarding matches on the ATP tour form 2017 to 2022. These observations have been cleaned to better organize variables and only include relevant variables.
- `atp_train.rds`: the training part of the split data
- `atps_test.rds`: the testing part of the split data
- `atp_split.rds`: the `atp` data following an initial split with strata on the `minutes` variable
- `keep_pred.rds`: the prediction technique used in model development
- `keep_wflow.rds`: the wflow used in model development
