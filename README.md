## Repository for Predictive Modeling of Duration of ATP Matches

Developing various models to predict the duration (in minutes) of matches on the ATP tennis tour.

### What's in this repository?

In this project, I developed six different models to predict how long matches on the ATP tour would take, in minutes. The predictions of these models were then compared with each other and the actual durations of methods to determine the best performing model.

### Directories

- `cleaned_data`: folder containing cleaned, prepared data used in this report
- `data`: folder containing raw data used in this report
- `figures`: folder containing figures developed in this report
- `memos`: folder containing progress memos created in this report
- `R Scripts`: folder containing R scripts used in this report to host various code
- `recipes`: folder containing recipes developed in this report
- `results`: folder containing the results of models' performances
- `tables`: folder containing tables developed in this report
